mysql之密码学

一、mysql命令行重置密码

1.1.首先需要连接进入mysql

1.2.重置新的密码

让它生效

二、设置mysql数据库root帐号密码

mysqladmin -u root password '新密码'

三、 为root账户设置密码

mysql_secure_installation
回车，根据提示输入Y
输入2次密码，回车
根据提示一路输入Y
最后出现：Thanks for using MySQL!
MySql密码设置完成，重新启动 MySQL：

/etc/init.d/mysqld restart #重启
/etc/init.d/mysqld stop #停止
/etc/init.d/mysqld start #启动

四、mysql忘记密码的重置方法

4.1.编辑mysql主配置文件 my.cnf

在[mysqld]字段下的位置添加参数
skip-grant

4.2.重启数据库服务

4.3.这样就可以进入数据库不用授权了

# mysql -uroot

4.4.修改相应database的用户密码

4.5.修改/etc/my.cnf 去掉 skip-grant , 重启mysql服务

# /etc/my.cnf
# service mysqld restart

4.6.番外

ps：同理也可以更改其他的 如wordpress、discuz库的密码：

1，使用mysql -u -p密码，进入mysql;
2，use wordpress；
3，update wp_users set user_pass=MD5(‘新密码’) where wp_users.user_login=’用户’;
4，flush privileges;
5，退出mysql 并重启mysqld服务
