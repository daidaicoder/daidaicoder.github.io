virtualbox之ubuntu全屏显示

2016-11-02 /
  • ubuntu
  • 原创
 / 点击 / 阅读耗时 1 分钟

在virtualbox安装好ubuntu后，显示的界面是800*600的分辨率，下面给出需要全屏显示的解决办法：

1.启动Ubuntu后，安装Virtualbox增强功能（在虚拟机的“设备”菜单里可以点击）

正在安装中。

2.安装增强功能完成后，选中虚拟机菜单上的“视图” -> “自动调整显示尺寸”。

3.点击虚拟机菜单上的“视图” -> “显示全屏”(快捷键：右Ctrl + F)

4.重启系统后就可以了，记得需要点一下虚拟机中“视图”下的“无缝模式”和“视图”下的“全屏模式”，此时缩小放大虚拟机窗口就会正常的变了。

全文完。
