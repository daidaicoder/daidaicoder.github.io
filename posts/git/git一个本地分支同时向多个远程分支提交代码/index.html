一、描述

本文网上学习记录，by daizelai on 2021/12/20 10:56

一个项目同时绑定多个分支，比如同时向码云和gitlab提交。

二、开始

2.1.方法一、添加多个远程仓库

项目根路径中右击gitbash，添加一个远程仓库

git remote add origin https://git.oschina.net/Jicklin/bokeyuan.git

这里的origin是仓库的名称 可以换成其它的。 然后再添加一个或多个：

git remote set-url --add origin https://github.com/jicklin/bokeyuan.git

如果是修改仓库地址使用如下命令：

git remote set-url origin https://github.com/jicklin/bokeyuan.git

这个时候就已经把远程仓库都添加进去了，把开.get/config可以发现：

[remote "origin"]
    url = http://gitlab.gbcom.com.cn/daizelai/sh18_android_app.git
    fetch = +refs/heads/*:refs/remotes/origin/*
    url = git@gitlab.gbcom.com.cn:daizelai/sh18_android_app.git

2.2.方法二、直接修改config配置

将远程仓库地址按照格式添加到config文件即可。 然后就可以同时同步两个远程仓库了。

$ git push origin master
Everything up-to-date
Everything up-to-date
