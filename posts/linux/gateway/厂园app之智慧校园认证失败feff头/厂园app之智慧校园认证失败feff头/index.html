一、认证失败之问题修复

1.通过日志找到认证地址：

authUrl-----http://api.gwifi.com.cn/cgi/api/token?phone=13735923202&pwd=e10adc3949ba59abbe56e057f20f883e&secret=e55185dd80decc7
b957da57d300cc727

2.直接在浏览器中输入地址返回也是对的。

{"response":100,"message":"success","data":{"accessToken":"e1fb9ead374c9a679e2a8e38f748f966a66f6002","expires":14400}}

3.通过打印日志发现，厂园用户认证平台返回数据多了一个<feff>头，导致认证失败。

4.下面进行判断删除，最好是厂园那边也做处理。

private function doSSOAuth($result, $username, $password, $orgId){

                    $loginReturn = substr($loginReturn ,3);
                }

二、获取用户信息失败之问题修复

1.这里问题解决了，再次点击智慧校园项目，发现提示“获取用户信息失败！”

 getUrl-----http://api.gwifi.com.cn/cgi/user/getUser?info=1&phone=13735923202&accessToken=6c9390cd30e888dc5b29531bcb29c324fb0aeb4c

2.应该是和刚才一样的错误，于是直接打印日志，一看果然如此，还是多了一个<feff>。

 userReturn-----<feff>{"response":100,"message":"success","data":{"id":"1219035","identityName":"\u8303\u4e00","orgId":"499","co
llegeId":"0","identityType":"1"}}

private function getUserInfoFromGwifi($username, $accessToken, $orgId){

                $userReturn = $this -> doSSOApi($getUrl);
