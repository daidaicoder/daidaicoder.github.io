一、获取厂园网关IP地址

1.1.使用串口连接网关

通常情况下可以这样连接，一根网线连接网关的WLAN接口，另一头连接网络，另一根网线连接网关网口，另一头连接到笔记本，此时得到IP后，就可以通过SSH连接网关访问了。

因为这里查询后，还是不不知道IP地址，不能通过SSH直接连接大网关，所以使用串口连接网关，CGW3000T网关正面最左边第一个网口是串口，第二个网口是WLAN外网接口。 使用串口线把网线口接入第一个网口，然后另一端接口连接台式机，并在win7系统的管理工具中查看使用的是哪个com端口。

使用SSH连接时，Protocol就不能选择SSH2了 ,选择Serial，然后看下在图片连接。

1.2.查看所有网卡

首先查看所有的网卡情况，把未激活的全部激活

-bash-4.1# ifconfig -a
-bash-4.1# ifconfig eth6 up // 激活eth6
-bash-4.1# ifconfig eth7 up // 激活eth7
-bash-4.1# ifconfig eth2.200 down // 禁用eth2.200

四、网桥配置命令brctl

网桥是一种在链路层实现中继，对帧进行转发的技术，根据MAC分区块，可隔离碰撞，将网络的多个网段在数据链路层连接起来的网络设备。

4.1.网桥安装

安装bridge-utils软件包，并加载bridge模块和开启内核转发。

# apt-get install bridge-utils
# modprobe bridge
# echo "1">/proc/sys/net/ipv4/ip_forward

4.1.常用命令

[root@CGW3000T htdocs]# brctl addif br0 eth0 // 让eth0成为br0的一个端口
[root@CGW3000T htdocs]# ifconfig br0 192.168.1.1 // 给br0的虚拟网卡配置IP：192.168.1.1。那样就能远程管理网桥。
[root@CGW3000T htdocs]# brctl --help // 帮助
[root@CGW3000T htdocs]# brctl addbr br0 // 增加网桥
[root@CGW3000T htdocs]# brctl delif br0 eth0 eth1 // 删除网桥

4.2.查看

[root@CGW3000T htdocs]# brctl show
bridge name     bridge id               STP enabled     interfaces
br0             8000.2851321091e3       no              eth1
                                                        eth2
                                                        eth3
br1             8000.2851321091e3       no              eth1.200
                                                        eth2.200
                                                        eth3.200

4.3.关闭网桥命令

# brctl delif ena eth1;
# brctl delif ena eth0;
# ifconfig ena down;
# brctl delbr ena;
