四、gpio查找

gpio查找其实也是在文件内容中查找。

# grep -rn "gpio99" kernel/msm-3.18/arch/arm64/boot/dts/qcom/
# grep -rn "tlmm 99" kernel/msm-3.18/arch/arm64/boot/dts/qcom/
# grep -rn "&msm_gpio" kernel/msm-3.18/arch/arm64/boot/dts/qcom/

五、文件查找

查找

# find . -name Android.mk

按文件类型查找：

# find ./device/qcom/ -type f -name splash.img
./device/qcom/common/display/logo/splash.img
./device/qcom/common/display/gbcom/splash.img
#

示例：

# find ./vendor/ -type f -name generate_extra_images.mk 
./vendor/qcom/build/tasks/generate_extra_images.mk
#
